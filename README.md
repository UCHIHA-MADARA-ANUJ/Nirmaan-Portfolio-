## 🛠️ Technical Stack &amp; Constraints Secured

* **Framework:** Next.js 14 (App Router) + TypeScript Strict + Tailwind CSS
* **Motion Trinity:** GSAP (`gsap`, `ScrollTrigger`), Studio Freight Lenis (`lenis`), Framer Motion (`framer-motion`), and Anime.js (`animejs`)
* **3D / WebGL:** Three.js (`three`), React Three Fiber (`@react-three/fiber`), React Three Drei (`@react-three/drei`)
* **Geospatial &amp; Data:** React Leaflet (`react-leaflet`, `leaflet`), Recharts (`recharts`)
* **State Management:** Zustand with local storage persistence (`zustand`)
* **Quality Assurance:** `npm run build` verified completely zero-error exit codes with pristine static generation.

## 💻 Running the Platform Locally

Execute the following commands from your terminal:

```bash
# Navigate to the repository
cd nirman-portfolio

# Install required packages (already completed in workspace)
npm install --legacy-peer-deps

# Launch the ultra-high-performance Next.js development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to engage the digital alchemy.
